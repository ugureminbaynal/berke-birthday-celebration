
Berke Herdem — Birthday Microsite (Flame/Graphite Trichromatic)

1) Purpose
A single-page, fast-loading greeting site to celebrate Berke’s birthday. Dark-first design with a Flame accent and a cool Cyan trace for links/highlights.

2) Color Tokens (Trichromatic)
--flame: #E0453A        (Primary / CTA / highlight)
--graphite: #111827     (Background / headings)
--mist: #E5E7EB         (Text on dark, borders)
--ink: #0F172A          (Alternate dark surface)
--cyan-trace: #22D3EE   (Links, small accents)

Dark Mode (default):
bg: var(--graphite)  (#111827)
surface: #0F172A
text: var(--mist)    (#E5E7EB)
primary: var(--flame)
accent: var(--cyan-trace)
divider: #1F2937

Light Mode:
bg: #F8FAFC
surface: #FFFFFF
text: #0F172A
primary: var(--flame)
accent: var(--cyan-trace)
divider: #E5E7EB

3) Typography
Display/Headings: Sora, 700/600
Body: Inter, 400/500
Code/Mono (if needed): JetBrains Mono, 500
Scale: 14/16/20/24/32/48px

4) Spacing & Corners
Spacing scale: 8, 12, 16, 24, 32, 48
Container width: max 720px
Card radius: 16px, subtle shadow
Animation: 180–220ms ease-out; hover raises by 2px

5) Components
- Primary Button:
  bg: var(--flame); text: #fff; hover: filter brightness(0.95); focus ring: 2px var(--cyan-trace)
- Link: color var(--cyan-trace); underline on hover
- Chip/Tag: bg rgba(224,69,58,0.14); text var(--mist); border 1px rgba(229,231,235,0.08)
- Divider: 1px solid divider token

6) Page Structure (Single page)
[Hero]
  - Badge: “HBD, Engineer!” (chip)
  - H1: “Mutlu Yıllar Berke!”
  - Sub: “Flame kadar sıcak, Graphite kadar sağlam bir sene.”
  - CTA1: “Mesaj bırak” → mailto
  - CTA2: “Projelerini gör” → GitHub link (placeholder)
  - Background motif: faint PCB traces at 4% opacity

[Section: Timeline/Highlights]
  - Title: “Bugün için küçük notlar”
  - 3 items (icons via Lucide):
    • “Yeni yaş = yeni devre.”
    • “Hedef: daha az latency, daha çok keyif.”
    • “Sağlık, huzur ve 10/10 verim.”

[Section: Gallery (optional)]
  - 3 photo slots; grayscale by default, on hover warms with Flame tint

[Footer]
  - Tiny text: “Seviliyorsun. — ekibin”
  - Links: Mail, LinkedIn, GitHub

  --flame:#E0453A; --graphite:#111827; --mist:#E5E7EB; --ink:#0F172A; --cyan-trace:#22D3EE;
}
@media(prefers-color-scheme:dark){
  :root{ --bg:#111827; --surface:#0F172A; --text:#E5E7EB; --divider:#1F2937; }
